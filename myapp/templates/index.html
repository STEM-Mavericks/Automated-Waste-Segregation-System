Project Details

Smart Dustbin System

Our Smart Dustbin System automatically sorts wet and dry waste using moisture sensors. This system is designed to enhance recycling efficiency and promote sustainable waste disposal practices. With load sensors and a user-friendly web interface, this project aims to reduce landfill usage and raise awareness about proper waste disposal.

The system also features an LCD display showing the waste weight, helping users track their daily waste production and recover misplaced items.
